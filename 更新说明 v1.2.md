# 排课管理系统 v1.2 完整更新说明

## 🎉 v1.2 新增功能（2026-04-10）

### 1. 多维度收入统计 ✅
- **日期范围统计**：自由选择统计时间段
- **按班型统计**：一对一/一对二/小组课/大班课的收入和占比
- **按课程来源统计**：自有课程/机构排课/混合班的收入和占比
- **按是否上门统计**：中心内上课 vs 上门教学
- **按机构统计**：各合作机构的收入贡献
- **月度趋势图**：一键生成收入趋势曲线图
- **班型占比饼图**：可视化显示各类型课程占比

### 2. 学生学费统计 ✅
- **按学生统计**：每个学生在统计期间的总学费
- **按班型分解**：每个学生在不同班型上的学费分布
- **自动计算**：一对多课程自动按人数分摊

### 3. 一对多课程定价优化 ✅
- **学生单独定价**：一对二/小班课/大班课支持对每个学生设置不同学费
- **课时费计算方式**：
  - 按一次课计算：老师课时费固定（如 200 元/次）
  - 按学生分摊：课时费按学生数累加（如 100 元/学生×3 人=300 元）

### 4. 上门服务类型 ✅
- **课程设置**：可选择课程是中心内上课还是上门教学
- **统计筛选**：收入统计可按是否上门维度分析

## 📦 安装说明

### 方法一：完整项目包（推荐）

1. **解压项目包**
   ```
   解压 scheduling-system-v1.2-complete.zip 到任意目录
   ```

2. **安装依赖**
   ```bash
   # 双击运行
   install.bat
   ```

3. **测试运行**
   ```bash
   npm run dev
   ```

4. **打包成 exe**
   ```bash
   # 双击运行
   build.bat
   ```

5. **安装应用**
   ```
   安装包位置：dist/排课管理系统 Setup 1.2.0.exe
   双击安装即可
   ```

### 方法二：更新现有项目

如果你已有 v1.1 版本：

1. **备份数据**
   - 打开旧版本
   - 系统设置 → 导出全部数据
   - 保存 JSON 备份文件

2. **替换文件**
   替换以下文件到项目目录：
   ```
   src/types/index.ts
   src/utils/helpers.ts
   src/services/browserDatabase.ts
   src/pages/RevenueStatistics.tsx (新增)
   src/App.tsx
   package.json
   ```

3. **重新安装依赖**
   ```bash
   rm -r -fo node_modules

   ```

4. **重新打包**
   ```bash
   build.bat
   ```

5. **导入数据**
   - 安装新版本
   - 系统设置 → 导入数据

## 🆕 新功能使用说明

### 1. 收入统计

**入口**：财务管理 → 收入统计

**操作步骤**：
1. 选择日期范围（如 2026-01-01 ~ 2026-12-31）
2. 自动显示总收入、课程数等统计
3. 点击图表按钮查看可视化：
   - 班型收入图（柱状图）
   - 来源占比图（饼图）
   - 月度趋势图（折线图）

**统计维度**：
- 按班型：一对一/一对二/小组课/大班课
- 按来源：自有课程/机构排课/混合班
- 按服务：中心内/上门
- 按机构：各合作机构
- 按月份：月度趋势

### 2. 学生学费统计

**位置**：收入统计页面底部

**显示内容**：
- 学生姓名
- 总学费（统计期间）
- 按班型分布（一对一多少、小班课多少等）

**计算逻辑**：
- 一对一：全额计入
- 一对二：学费÷2 计入每个学生
- 小班课：学费÷学生数 计入每个学生

### 3. 一对多课程定价

**课程设置时**：
1. 选择课程类型为一对二/小组课/大班课
2. 填写基础学费和课时费
3. 选择课时费计算方式：
   - 按一次课：老师固定收入
   - 按学生分摊：按人数累加

**排课时**：
1. 选择课程
2. 选择上课学生（可多选）
3. 系统自动计算：
   - 学费 = 课程学费 × 时长系数
   - 课时费 = 根据计算方式自动汇总

## ❓ 常见问题

**Q: 统计数据不准确？**
A: 确保课程状态标记为"已完成"，只有完成的课程才会计入收入统计。

**Q: 一对多课程学费如何分摊？**
A: 系统默认按学生人数平均分摊，可在排课时手动调整。

**Q: 图表不显示？**
A: 检查是否安装了 chart.js 和 react-chartjs-2 依赖。

**Q: 如何导出统计报表？**
A: 截图保存，或使用浏览器的打印功能导出 PDF。
