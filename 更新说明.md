# 排课管理系统 v1.1 更新说明

## 🎉 新增功能

### 1. 年级信息优化
- ✅ 增加"入学年份"字段（如 2025 级）
- ✅ 自动计算当前年级（高一/高二/高三）
- ✅ 每年 9 月自动升级

### 2. 课程表界面优化
- ✅ 固定宽度适配屏幕（无横向滚动条）
- ✅ 高度自适应（最少 5 行，根据课程数量调整）
- ✅ 左侧不显示时间列，节省空间
- ✅ 课程内容居中显示（课程名 + 地址一行，时间一行）

### 3. 学费与课时费区分
- ✅ 课程设置：学费（学生交费）+ 课时费（老师到手）
- ✅ 计费单位：按小时 / 按次课（可选）
- ✅ 排课时自动计算（按时长×单价）

### 4. 机构课程管理
- ✅ 机构排课增加机构目录
- ✅ 可记录机构名称、联系人、分成比例

### 5. 打包成 exe 安装包
- ✅ 使用 electron-builder 打包
- ✅ 生成 setup.exe 安装文件
- ✅ 支持自动更新（后续版本）

---

## 📦 打包说明

### Windows 用户

1. **双击运行** `build.bat`
2. 等待打包完成（约 2-5 分钟）
3. 安装包位置：`dist/排课管理系统 Setup 1.1.0.exe`

### 手动打包

```bash
# 安装依赖
npm install --legacy-peer-deps

# 构建应用
npm run build

# 打包 Windows 安装包
npm run dist:win
```

---

## 🚀 安装说明

1. 双击 `排课管理系统 Setup 1.1.0.exe`
2. 选择安装位置（或默认）
3. 完成安装
4. 桌面会自动创建快捷方式

---

## 📝 升级说明

**从 v1.0 升级到 v1.1：**

- 旧版本数据会自动迁移（存储在 LocalStorage）
- 建议先导出备份（系统设置 → 导出全部数据）
- 安装新版本后，导入旧数据即可
